The Silent Shape

Game Description:

This is a game based of a game that was trending a while ago called "Granny", in this game the player needs to collect items to craft distractions that can be used draw the demon doll's attention away from you so that you can get the key to the house and escape.

My Role:

For this project i was the only person contributing so i did the art and assets, the code, the ui, the sound i got off the internet and coded it in.
